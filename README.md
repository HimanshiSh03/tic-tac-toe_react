Tic-Tac-Toe Game 
A simple 2-player Tic Tac Toe game built using ReactJS.
Features
- Classic 3x3 Tic-Tac-Toe board
- 2-player turn-based logic
- Win/draw detection
- Clean and responsive UI
Tech Stack
- ReactJS
- HTML, CSS, JavaScript
Live Demo

Run Locally
```bash
git clone https://github.com/HimanshiSh03/tic-tac-toe_react.git
cd tic-tac-toe_react
npm install
npm run dev
